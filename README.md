# Portfolio

Ce projet est un portfolio interactif conçu pour mettre en valeur vos compétences, vos projets et votre parcours professionnel. Il offre une expérience utilisateur fluide et moderne, avec un design élégant et des animations captivantes.

---

## Démo en ligne
👉 [Lien vers la démo](https://Alex05510.github.io/Portfolio/)

---

## Technologies utilisées
- **HTML** : Structure des pages.
- **CSS** : Styles et mise en page.
- **SCSS** : Préprocesseur CSS pour des styles modulaires et organisés.
- **JavaScript** : Interactivité et logique.


---

## Fonctionnalités principales
1. **Sections personnalisées** :
   - **À propos de moi** : Présentation personnelle, compétences, et passions.
   - **Mes projets** : Une galerie interactive présentant vos projets, avec descriptions et liens.
   - **Contact** : Formulaire de contact fonctionnel permettant de vous envoyer des messages.

2. **Design réactif** :
   - Optimisé pour tous les appareils (ordinateurs, tablettes et mobiles).

3. **Animations GSAP** :
   - Transitions fluides entre les sections.
   - Effets visuels modernes pour enrichir l'expérience utilisateur.
